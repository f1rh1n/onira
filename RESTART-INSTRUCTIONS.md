# How to Restart Onira

## Quick Fix

1. **Close this terminal/command prompt**

2. **Double-click** `cleanup-and-start.bat` in the bakery folder

3. **Wait** for the server to start (will show "Ready in X seconds")

4. **Open browser** to http://localhost:3000

---

## Manual Fix (if batch file doesn't work)

1. Open Task Manager (Ctrl + Shift + Esc)
2. Find all "Node.js JavaScript Runtime" processes
3. Right-click each one and select "End Task"
4. Open a fresh terminal in the bakery folder
5. Run: `npm run dev`
